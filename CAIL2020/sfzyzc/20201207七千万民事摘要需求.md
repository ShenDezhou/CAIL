# 摘要运算需求

# 1. 摘要模型
第二版摘要模型位于192.168.60.15:58096上，按照法宝线上案由分类标准，增加了支持的案由类型，从6类增加到了128类。
在运算速度上，提高了单词运算效率，20秒左右出一个运算结果。

## 1.1 接口请求
请求服务器和端口为192.168.60.15:58096  
调用方法为HTTP的POST方法    
请求地址：/z  
请求数据格式为json格式，示例如下：  
``
{
    "1":"some gid",
    "2":"民事正文文本"
}
``
